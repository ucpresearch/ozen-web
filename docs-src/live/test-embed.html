Usage Instructions

Setting Up Test Files

To test these examples:

  1. Create directory: mkdir -p static/test-samples
  2. Add test audio files (WAV format recommended):
    • short.wav - Short sample (5-10 seconds)
    • long.wav - Long sample (>60 seconds, optional)
  3. Start dev server: npm run dev
  4. Open: http://localhost:5173/test-embed.html

Embedding in Your Own Site

Option 1: Relative paths (portable) - Works from any directory:

<!-- HTML file in same directory as viewer.html -->
<iframe
  src="viewer.html?audio=test-samples/short.wav&overlays=pitch,formants,hnr"
  width="100%"
  height="600"
  frameborder="0">
</iframe>

Option 2: Absolute paths - Requires knowing deployment location:

<!-- Deployed at domain root -->
<iframe
  src="/viewer.html?audio=test-samples/short.wav&overlays=pitch,formants,hnr"
  width="100%"
  height="600"
  frameborder="0">
</iframe>

<!-- Deployed at /resources/ozen/ subdirectory -->
<iframe
  src="/resources/ozen/viewer.html?audio=test-samples/short.wav&overlays=pitch,formants,hnr"
  width="100%"
  height="600"
  frameborder="0">
</iframe>

Option 3: Full URL - For embedding from external sites:

<iframe
  src="https://yoursite.com/resources/ozen/viewer.html?audio=https://cdn.example.com/audio.wav&overlays=pitch,formants,hnr"
  width="100%"
  height="600"
  frameborder="0">
</iframe>
